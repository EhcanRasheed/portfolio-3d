About

I bring strong critical thinking and solid foundations across software architecture, system design and core programming, paired with a genuine curiosity for the corners of computing most developers never explore. I believe it all connects in the end.

My work spans the full development cycle. I architect and build complete web applications with React, Next.js and Node.js, write clean backend logic, and make sure everything that ships is fast, scalable and built to last. I'm fluent across Python, C++, C# and .NET too, and I reach for whatever the problem actually needs rather than defaulting to a favourite stack out of comfort.

On the AI side, I don't just wire up an API and call it intelligent. I build and train models. For a house price predictor aimed at the Islamabad market, clean local data simply didn't exist, so I generated and calibrated my own from scratch. That ground-up, figure-it-out instinct is the thread through everything I build. A background in security also means I think about how a system breaks before I think about how it ships. If you're working on something ambitious, I'm interested.
